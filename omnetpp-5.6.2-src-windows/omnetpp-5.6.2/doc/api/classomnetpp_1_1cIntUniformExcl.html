cIntUniformExcl Class Reference
Random Number Generation » Discrete Distributions

#include <crandom.h>

Description

Generates random numbers from the discrete uniform distribution.

This class is an object-oriented wrapper around the intuniformexcl(cRNG *rng, int a, int b) function.
