cLongHistogram Class Reference
Statistical Result Collection

#include <clegacyhistogram.h>

Description

Equidistant histogram for integers.

DEPRECATED CLASS, use cHistogram instead.

This class is just a cLegacyHistogram preconfigured for collecting integers, that is, it operates in INTEGERS mode.
