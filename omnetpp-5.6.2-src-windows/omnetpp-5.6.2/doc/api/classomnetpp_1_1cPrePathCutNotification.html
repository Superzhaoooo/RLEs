cPrePathCutNotification Class Reference
Signals

#include <cmodelchange.h>

Description

Model change notification fired just before a connection path is cut.

This notification is fired at the top of cGate::disconnect() on the owner modules of the start AND end gates of the connection path that is about to be cut when the gate gets connected. See cPathChangeNotification for more details.

This object accompanies the PRE_MODEL_CHANGE signal.
