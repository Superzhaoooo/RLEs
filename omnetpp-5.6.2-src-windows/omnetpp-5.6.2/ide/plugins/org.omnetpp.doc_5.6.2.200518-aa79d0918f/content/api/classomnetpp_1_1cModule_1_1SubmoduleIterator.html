cModule::SubmoduleIterator Class Reference

#include <cmodule.h>

Description

Iterates through the submodules of a compound module.

Usage:

for (cModule::SubmoduleIterator it(module); !it.end(); ++it) {
cModule *submodule = *it;
...
}
